# AutoAdvocate

**Know before you pay.**

A Next.js prototype that helps car owners understand trouble codes and question repair recommendations before spending money.

## Included

- Professional responsive landing page
- Dashboard
- Vehicle profile form
- OBD-II code analyzer with P0302, P0171, and P0420
- Repair estimate review prototype
- Mechanic rules supplied by the founder

## Run locally

1. Install Node.js 20 or later.
2. Open a terminal in this folder.
3. Run `npm install`
4. Run `npm run dev`
5. Open `http://localhost:3000`

## Deploy

Import the GitHub repository into Vercel and deploy using the default Next.js settings.

## Important

This prototype provides educational guidance and is not a substitute for an in-person inspection by a qualified technician.
